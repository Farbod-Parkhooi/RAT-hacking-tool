# installing

## First, you should install <a href="https://github.com/Unknow-per/Its_Hub-Library/"> Its_Hub Library </a>.

```
git clone https://github.com/Unknow-per/MS-TL/

cd MS-TL

pip install -r requirements.txt

python setup.py
```
